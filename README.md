- Pendiente

- Pendiente

## Cómo ejecutar
1. Abre la carpeta `Registro-de-Gastos` en Visual Studio o VS Code.
2. Verifica que el dispositivo Android esté conectado y visible en ADB.
3. Ejecuta `dotnet build` para compilar el proyecto.
4. Presiona `F5` o usa `dotnet run -f net10.0-android` para desplegar la app en Android.

## Notas
